## Why

时间线页面的日期选择器存在三个 UI 问题影响用户体验：
1. 展开时有 `OVERFLOWED BY 0.4 PIXELS` 的溢出警告
2. 展开后选中日期的文字颜色为白色，与背景对比度不足
3. 展开后收回时，周视图会跳转到今天所在的周，而非选中日期所在的周

这些问题影响用户正常使用日期选择功能，需要修复。

## What Changes

- **修复溢出问题**：调整周视图日期单元格的布局，消除 0.4 像素的溢出
- **修复选中日期颜色**：在月视图（展开状态）下，为选中日期添加明显的背景样式，确保白色文字可读
- **修复收回定位问题**：收回周视图时，保持显示选中日期所在的周，而非自动跳转到今天所在的周

## Capabilities

### New Capabilities

无新增能力。

### Modified Capabilities

- `timeline-date-picker`: 修复日期选择器的溢出、颜色和定位问题

## Impact

- **影响文件**：`lib/widgets/timeline/date_picker.dart`
- **影响范围**：时间线页面的日期选择器组件
- **无 API 变更**：仅修复 UI 问题，不影响对外接口
