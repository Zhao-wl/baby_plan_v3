## Context

E.A.S.Y. 育儿助手是一个基于 Flutter 的跨平台移动应用，需要支持 Web、Android 和 iOS 三端。项目已有完整的产品需求文档（PRD v1.1）和界面原型设计，现在需要制定可执行的技术实施方案。

当前状态：
- 技术栈已初步确定：Flutter 3.x + Dart
- 需要详细设计数据层、服务层、状态管理层和表现层
- 需要针对中国网络环境进行云服务选型和验证
- 需要制定详细的开发排期和里程碑计划

约束条件：
- 必须支持本地优先（Local-First）架构，断网可用
- 必须适配中国网络环境（云服务、第三方登录等）
- 需要家庭组协同功能，支持多设备数据同步
- 需要预测算法支持智能行为预测

## Goals / Non-Goals

**Goals:**
- 建立完整的技术架构设计文档
- 完成核心技术选型并制定验证方案
- 制定 10 周开发排期，明确任务依赖和并行关系
- 定义 5 个里程碑及验收标准
- 建立打包流程验证机制（M1 空包验证 + M4 正式验证）
- 识别技术风险并制定应对策略

**Non-Goals:**
- 不涉及具体的 UI 组件实现细节
- 不涉及后端服务代码编写
- 不涉及应用商店上架后的运营策略
- 不覆盖 iOS 企业证书相关配置

## Decisions

### 1. 本地数据库：Isar 3.1

**选择理由：**
- 类型安全，与 Dart/Flutter 深度集成
- 性能优秀，支持复杂查询和索引
- 自动迁移，减少 schema 变更带来的维护成本
- 支持同步和异步操作，适合本地优先架构

**备选方案：** SQLite/Drift - 生态成熟但类型安全较弱

### 2. 状态管理：Riverpod 2.4+（代码生成版）

**选择理由：**
- 编译时安全，避免运行时错误
- 依赖注入友好，测试更容易
- 支持代码生成，减少样板代码
- 与 Flutter 官方团队维护，长期支持有保障

**备选方案：** Bloc - 学习曲线较陡，样板代码较多

### 3. 云服务：UniCloud（DCloud）优先，Laf 备选

**选择理由：**
- 国内节点，延迟低（<100ms）
- 无需备案即可使用基础功能
- 支持微信登录（需企业资质，个人开发者可用手机号替代）
- 免费额度满足初期需求

**验证重点：**
- 国内网络连通性
- 函数冷启动时间
- 数据库连接稳定性

**备选方案：** Laf - 开源 Serverless，部署灵活

### 4. 图表库：fl_chart 0.68

**选择理由：**
- 生态成熟，社区活跃
- 支持自定义样式，可实现设计稿中的效果
- 性能满足需求

**验证重点：**
- 堆叠柱状图（睡眠分布）
- 折线图（生长曲线）
- 饼图（E.A.S.Y 循环分析）

## Risks / Trade-offs

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| 云服务（UniCloud/Laf）不稳定或无法通过验证 | 高 | 第1周立即验证，准备自建服务器备选方案 |
| fl_chart 无法满足设计稿中的图表效果 | 中 | 第5周前完成图表验证，必要时使用 CustomPainter 自定义绘制 |
| iOS 开发者账号审核或证书配置问题 | 中 | M1 阶段即进行 iOS 模拟器构建验证 |
| 预测算法效果不达预期 | 低 | MVP 使用规则引擎，后续迭代引入机器学习 |
| 跨平台兼容性问题（尤其是 Web 端） | 中 | 每周在真机上测试，不只是模拟器 |

**Trade-offs：**
- 使用 Isar 而非 SQLite：获得类型安全但增加依赖体积（约 2-3MB）
- 使用 Riverpod 代码生成版：减少运行时错误但增加构建步骤
- 优先本地通知而非推送服务：降低复杂度但减少实时性

## Migration Plan

本项目为新建项目，不涉及迁移。部署策略：

1. **M1（第2周）**：空包打包验证
   - Android：`flutter build apk --release`
   - iOS：`flutter build ios --release`（模拟器）
   - Web：`flutter build web --release`

2. **M4（第8周）**：正式签名打包验证
   - Android：配置正式签名 keystore
   - iOS：配置 Apple Developer 证书
   - Web：PWA 配置验证

3. **M5（第10周）**：应用商店准备
   - 准备截图、描述、隐私政策
   - 提交各平台审核
