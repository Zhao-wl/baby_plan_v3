## Context

当前时间线页面已实现日期选择器组件（`lib/widgets/timeline/date_picker.dart`），使用 PageView 实现横向滑动切换周。主要组件包括：

- `WeekModel`: 管理一周日期数据
- `TimelineDatePicker`: PageView + 周视图
- `_buildHeader()`: 显示年月标题，点击调用 `showDatePicker()`

**当前状态**：
- ✅ 周视图滑动切换正常工作
- ✅ 点击日期选中功能正常
- ✅ 今天标识和选中高亮正常
- ❌ 月份标题不随滑动更新（总是显示 selectedDate 的月份）
- ❌ 点击月份弹出系统日历，交互割裂

**技术约束**：
- 使用 Flutter 动画系统实现展开/收起
- 保持 PageView 滑动手势不冲突
- 遵循 Material 3 设计规范

## Goals / Non-Goals

**Goals:**
- 滑动周视图时，月份标题实时显示当前可见周的月份
- 点击月份标题原地展开为月历视图
- 月历视图支持左右滑动切换月份
- 展开动画流畅自然
- 保持原有周视图选择功能

**Non-Goals:**
- 不改变时间线列表组件
- 不修改 Provider 数据层
- 不添加新的日期范围选择功能
- 不涉及深色模式特殊处理

## Decisions

### 1. 月份显示：基于可见周计算

**选择**：使用当前 PageView 可见页对应的周来计算月份，而非 selectedDate。

**理由**：
- 用户滑动浏览时，期望看到的是当前屏幕上日期所在的月份
- 选中日期和浏览位置可能不同（用户可以浏览其他周但不选中）
- 符合主流日历应用的交互习惯

**实现**：
```dart
// 在 onPageChanged 回调中更新当前显示的月份
String get _currentDisplayMonth {
  final weekStart = _getWeekStartForPage(_currentPage);
  // 获取周中间日期（周四）作为月份参考
  final middleDay = weekStart.add(Duration(days: 3));
  return '${middleDay.year}年${middleDay.month}月';
}
```

### 2. 展开/收起：AnimatedSize + 状态切换

**选择**：使用 AnimatedSize 实现高度动画，内部切换周视图/月视图。

**理由**：
- AnimatedSize 自动处理高度变化动画
- 不需要手动管理 AnimationController
- 与 Material 3 设计风格一致

**实现方案**：
```
收缩状态（周视图）：高度 120px
  - 年月标题（可点击）
  - 一行日期（7天）

展开状态（月视图）：高度约 280px
  - 年月标题 + 左右箭头
  - 星期标题行
  - 4-6行日期网格
```

### 3. 月历视图：PageView + 动态行数

**选择**：月历视图也使用 PageView 实现，支持左右滑动切换月份。

**理由**：
- 与周视图交互一致
- 支持无限滑动
- 可以复用 PageView 的预加载机制

**实现细节**：
- 每页显示一个月的日期（按周排列，最多6行）
- 切换月份时更新标题
- 保持选中日期高亮

### 4. 手势处理：点击 + 滑动分离

**选择**：
- 点击月份标题：切换展开/收起状态
- 收起时左右滑动：切换周
- 展开时左右滑动：切换月份
- 点击日期：选中日期并收起（展开状态下）

**理由**：
- 避免手势冲突
- 保持直观的操作逻辑
- 用户可以快速选择日期后自动收起

### 5. 状态管理：内部状态 + 回调通知

**选择**：展开/收起状态由组件内部管理，选中日期通过回调通知父组件。

**理由**：
- 展开/收起是 UI 状态，不需要外部管理
- 选中日期需要通知父组件刷新时间线
- 保持组件封装性

## Risks / Trade-offs

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| 动画卡顿 | 中 | 使用 AnimatedSize 而非手动动画，减少性能开销 |
| 月历行数不固定（4-6行） | 低 | 使用固定高度容纳6行，空白区域保持透明 |
| 手势冲突 | 低 | 展开时禁用周视图滑动，收起时禁用月视图滑动 |
| 大量日期计算 | 低 | 使用缓存和惰性计算 |

**Trade-offs：**
- 固定高度 vs 动态高度：选择固定高度（容纳6行），简化动画计算
- 周视图/月视图分离 vs 复用：选择部分复用（日期单元格），周/月容器分离
